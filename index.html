Projects

Master's Thesis

For my Master's Thesis, I worked, under the supervision of Pr. Peter Pietzuch, on the fascinating topic of Intel Software Guard eXtensions (SGX). Intel SGX ensures confidentiality and integrity of applications' key business code and data but incurs a sizeable performance penalty due to memory constraints. My goal was to minimise said penalty and find tradeoffs between confidentiality, integrity and performance. This project taught me a lot about research, time management, goal setting, low-level kernel driver programming (C), and a lot about the JVM's memory and execution model.

Automatic Air Traffic Control For Drones

  • The project in a nutshell: Team project in which we built a robust air traffic management algorithm which foresees collisions between drones, manned aviation, aerial and ground hazards and recommends actions and/or course corrections to prevent them. We worked on this project in association with Altitude Angels and Microsoft. See project documentation, demo tests, and press release.
  • In this project, I researched, implemented, benchmarked geographical areas representation in software, efficient and scalable graph algorithms for path-finding (A*, Dijkstra, θ*, Lazy θ*, Block-A*) and no-fly-zone detection in C#, that heavily contributed to the project’s success [c.f documentation #page 14].
